Resume Project Summary
                                                                                                                                                         
  Investment Transaction Processing & Lot Tracking System
                                                                                                                                                         
  Built event-driven data pipelines processing daily mutual fund transactions from multiple fund administrators. The system ingests heterogeneous file
  formats, normalizes 120+ transaction type codes into a unified schema, and maintains investor holdings using FIFO lot-based cost tracking.             
                  
  Key Contributions:
  - Developed multi-format parsers (DBF, delimited text) to ingest transaction feeds from third-party fund administrators, normalizing vendor-specific
  codes into standardized transaction types                                                                                                              
  - Built in-memory FIFO lot matching engine with O(log n) sorted insertion — creates purchase lots on buys, depletes lots on sells, generates matched
  lot records for realized gain/loss calculation                                                                                                         
  - Implemented capital gains computation engine supporting jurisdiction-specific tax rules: cost-basis adjustments, inflation indexation, grandfathering
   provisions, and holding period classification (short-term vs long-term)                                                                               
  - Designed cross-batch deduplication using content hashing to ensure idempotent processing across overlapping data feeds
  - Built corporate action matching (switches, reversals) to correctly pair related transactions                          
  - Optimized database writes using batch operations and aggregation pipelines — reduced query volume by 95%
                                                                                                                                                         
  Tech Stack: Python, Apache Airflow, Polars, MongoDB, Kafka, AWS S3
                                                                                                                                                         
  Domain: Fund administration operations, investment lot accounting, FIFO cost-basis tracking, capital gains tax compliance, multi-source data
  reconciliation
